# CyberJoar Electoral Intelligence Matrix (PS5)
**Multi-Dimensional Predictive Analytics Platform**

## Project Overview
The CyberJoar Electoral Intelligence Matrix is a premium, data-driven dashboard designed for high-fidelity political forecasting. It integrates multi-source intelligence, weighted factor modeling, and real-time turnout simulations to project electoral outcomes across diverse tactical environments.

## Technical Stack
- **Frontend**: React 18 + Vite
- **Styling**: Tailwind CSS (Premium Navy/Gold Theme)
- **Visualizations**: Recharts (Radar, Bar, Area charts)
- **Icons**: Lucide React
- **Logic**: Weighted Scoring Engine + Voter Turnout Bias (Anti-Incumbency Modeling)

## Key Intelligence Features
1. **Factor Weights Matrix**: Dynamically recalibrate 6 primary influence variables (Incumbency, Party Strength, Past Work, Personal Base, Demographic Alignment, OSINT Sentiment).
2. **Turnout Simulation**: Model voter participation scenarios (30%-95%) to observe how volatility and cadre mobilization shift the Probability of Win (PoW).
3. **Strategic Gap Analysis**: Automatically identifies asymmetric advantages and tactical vulnerabilities between candidate profiles.
4. **OSINT Sentiment Feed**: Integrated digital discourse tracking using pseudo-real-time social media and news intelligence.
5. **State Persistency**: Generate shareable intelligence URLs and export raw JSON/CSV reports for external synthesis.
