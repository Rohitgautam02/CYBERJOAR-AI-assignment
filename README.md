## 🚀 Deployment & Initialization

### Root Configuration
Install dependencies for all workspaces from the root:
```bash
npm install
```

### Direct Tactical Launch
Run individual units directly via root scripts:
```bash
# Launch Strategic Fusion (PS1)
npm run dev:ps1

# Launch Electoral Matrix (PS5)
npm run dev:ps5
```
